In order to run this code, first make sure you have a MySQL server installed.
Run the "databasedump.sql" file (for instance in MySQL workbench) to copy the database/schema over to your machine.

To run the user application, navigate to the Debug folder under the solution and click "Look Inna Book.exe"
To run the owner/admin application, use "LookInnaBook.bat"

Depending on your credentials, you may need to edit the connection string in the ConnectionString.txt file
